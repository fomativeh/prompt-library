### **Prompt: Text to Table Formatter**

**Objective:** Convert plain text into a structured table.

**Prompt:**
```
Convert the following text into a markdown table:
"Apples cost 2 dollars. Bananas cost 1 dollar. Oranges cost 3 dollars."
```

**Example Output:**

 | Fruit | Price |
 |-------|-------|
 | Apples | $2 |
 | Bananas | $1 |
 | Oranges | $3 |
